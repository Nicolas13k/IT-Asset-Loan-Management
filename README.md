# IT-Asset-Loan-Management
Internal IT asset loan tracking application

This project was developed during my internship to manage and track company IT equipment loan.

The application allows IT staff to:
- Track boroed devices
- View available devices
- Associate quipment with employees
- Monitor loan and return dates
- Manage IT inventory records

Technologies

- PHP
- MySQL
- HTML
- CSS
- PhphMyAdmin

The database contains three main entities:

Employee
- Employee ID
- First name
- Last name
- Department
- Leaving date

Device
- Device ID
- Type
- Brand
- Model
- Serial number
- MAC address

Loan
- Loan ID
- Employee ID
- Device ID
- Loan date
- Expected return date

- Database design
- SQL querying
- Web application development
- IT asset management
- Technical documentation
